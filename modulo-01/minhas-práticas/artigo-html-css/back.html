Back End

O que é Back-end ?

O back-end é a camada de desenvolvimento web que abrange linguagens de programação para se fazer a logica de aplicações web, as linguagens que são utilizadas nesta camada onde o usuario comum não consegue visualizar são uma combinação de servidores e bancos de dados, esta estrutura é onde os dados e as regras de negócios da empresa são processados e devolvidas para o front-end, onde o usuário final vê as informações na tela.

O que faz um desenvolvedor front-end ?

O desenvolvedor back-end deve conhecer bem uma ou algumas das linguagens de programação como PHP, Java, C# e Python, além de conhecimento em bancos de dados como o MySQL, para estruturar toda a lógica, regras de negócios, segurança, comunicação com bancos de dados e tratamento dos dados.
