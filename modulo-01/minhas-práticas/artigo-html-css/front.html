Front End

O que é Front-end ?

O front-end é o conceito de desenvolvimento web que abrange o HTML o CSS e o JavaScript como ferramentas para a construção de aplicações e sites web. Este conceito é classificado assim pois o desenvolvedor que atua como front-end designa suas habilidades tecnicas para construir os elementos da pagina ou sistema web, como textos, imagens, videos, estilização de cores, posicionamento, interação, validação de formulários e muitas outras configurações em que o usuario visualiza e interage.

O que faz um desenvolvedor front-end ?

O desenvolvedor front-end é responsável por estabelecer e esquematizar como cada elemento irá aparecer na tela do usuário. Então ele faz o uso das ferramentas e linguagens para isso deve ter o conhecimento em HTML, CSS e JavaScript para:

  • definir a estrutura das páginas;
  • demarcar conteudos específicos;
  • definir aonde conteudos devem aparecer;
  • estilizar a estrutura criada;
  • atribuir cores aos componentes;
  • definir posicionamento dos elementos;
  • validar informações de formulários;
  • fazer animações.
