Ferramentas de Desenvolvomento Web

HTML

HTML ou Linguagem de HiperTexto e Marcação é uma tecnologia web com intuito de apresentar marcações em textos e outras informações na internet.

Com ela é possível construir suas páginas web, e contribuir seus conhecimentos e suas informações com o mundo!. Pois ela permite que os navegadores entendam as marcações feitas que chamamos de tags, elas dão ênfase semântica ao conteúdo que digitamos em nossas páginas web.

O que podemos fazer com HTML

  • Demarcar o conteúdo de forma semântica.
  • Demarcar títulos de artigos e paragrafos.
  • Demarcar palavras que tenham ênfase em seus artigos.
  • Demarcar âncoras, ou links, que interligam seu conteúdo a outros artigos ou sites.
  • Demarcar tags que carregam imagens para seu site.

CSS

As CSS ou Folhas de Estilo em Cascata, é uma tecnologia que coopera com a HTML, ela é utilizada para fazermos nossas formatações de tipos de fontes, cores, espaçamentos de conteúdos e muito mais!. Ou seja com ela podemos criar os nossos proprios design de conteúdo, que será exibido para os leitores de nossos sites.

O que podemos fazer com CSS

  • Estilizar fontes e tamanhos de textos.
  • Mudar a cor de fundo das páginas web, e formato de elementos como blocos de conteudo.
  • Definir tamanho de imagens, e blocos de conteúdos dentro do site.
  • Definir a posição de conteúdo dentro das páginas web.

JavaScript

O JavaScript é uma linguagem de programação utilizada principalmente no desenvolvimento web, com ela podemos fazer interações nas nossas páginas junto com o HTML e o CSS, é interação junto com as outras ferramentas de desenvolvimento é denominada de front-end ou client-site, ou seja a camada de visualização dos sistemas e sites web.

Algumas coisas que podemos fazer com JavaScript

  • Mostrar ou esconder informações ao clicar em um botão
  • Montar carrosséis com transição de imagens
  • Dar zoom em imagens
  • Fazer animações
